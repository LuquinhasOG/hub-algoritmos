# SHA-256

Este projeto é uma implementação do Algoritmo de Hashing Seguro (SHA), que gera hashs de 256 bits, segundo especificações FIPS 180-4.
Este algoritmo apenas gera hash de textos, não servindo para gerar hash com arquivos binários;

O executável gerado por este código recebe um argumento, que é o diretório do arquivo contendo o texto para gerar o hash. Após o processamento, o hash é impressso no console.
